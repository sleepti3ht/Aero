# Aero

A small Windows utility for testing acrylic transparency with GradientColor.

## Features
- Modern compact UI
- Opacity slider
- Startup toggle

## Installation
Download the latest release and run `Aero.exe`.
Python and dependencies are not required.

## Run locally
```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Notes
- Settings are saved to `config.json`.
